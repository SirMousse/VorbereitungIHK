JavaScript — Event-Typen Übersicht

Alle wichtigen Events · Kategorie · Bubbling · Beispiel  |  Zum Abdrucken / Whiteboard
Maus-Events (9)
click
Linksklick auf ein Element
bubbeltabbrechbar
btn.addEventListener('click', (e) => { console.log(e.clientX, e.clientY); });
dblclick
Doppelklick
bubbelt
el.addEventListener('dblclick', (e) => { e.preventDefault(); // Selektion verhindern });
mousedown / mouseup
Taste drücken / loslassen · button: 0=L 1=M 2=R
bubbelt
el.addEventListener('mousedown', (e) => { if (e.button === 2) console.log('Rechtsklick'); });
mousemove
Maus bewegt sich · clientX / clientY
bubbelt
⚡ Sehr häufig — mit requestAnimationFrame drosseln!
mouseenter / mouseleave
Maus betritt / verlässt Element
kein Bubbling
Unterschied zu over/out: ignoriert Kind-Elemente
mouseover / mouseout
Maus über Element oder Kind
bubbelt
contextmenu
Rechtsklick / Kontextmenü öffnet
bubbeltabbrechbar
el.addEventListener('contextmenu', (e) => { e.preventDefault(); // eigenes Menü });
Tastatur-Events (3)
keydown
Taste wird gedrückt · wichtigste Methode
bubbeltabbrechbar
document.addEventListener('keydown', (e) => { if (e.key === 'Enter') submit(); if (e.ctrlKey && e.key === 's') save(); // e.repeat === true wenn Taste gehalten });
keyup
Taste wird losgelassen
bubbelt
input.addEventListener('keyup', (e) => { console.log('Wert:', e.target.value); });
keypress ⚠️
Taste gedrückt — VERALTET seit 2022
deprecated
Nicht mehr verwenden! keydown / keyup stattdessen.
Touch & Pointer (4)
touchstart
Finger berührt Screen · touches[0].clientX/Y
bubbelt
el.addEventListener('touchstart', (e) => { startX = e.touches[0].clientX; }, { passive: true }); // kein Ruckeln!
touchmove / touchend
Finger bewegt sich / verlässt Screen
bubbelt
pointermove
Maus / Touch / Pen — vereinheitlicht
bubbelt
// e.pointerType: 'mouse' | 'touch' | 'pen'
Formular-Events (7)
submit
Formular wird abgesendet
bubbeltabbrechbar
form.addEventListener('submit', (e) => { e.preventDefault(); // kein Seitenreload! const data = new FormData(e.target); });
input
Wert ändert sich — live bei jeder Eingabe
bubbelt
input.addEventListener('input', (e) => { preview.textContent = e.target.value; });
Unterschied zu change: feuert sofort, nicht erst beim Verlassen
change
Feld verlässt Fokus mit neuem Wert
bubbelt
Für select / checkbox → change · Für text → input
focus / blur
Fokus bekommen / verlieren
kein Bubbling
input.addEventListener('blur', () => validate(input.value));
focusin / focusout
Fokus – mit Bubbling (für Delegation)
bubbelt
form.addEventListener('focusin', (e) => { e.target.classList.add('active'); });
reset
Formular wird auf Standardwerte zurückgesetzt
bubbeltabbrechbar
Dokument & Lifecycle (5)
DOMContentLoaded
HTML fertig geparst — kein CSS/Bild nötig
kein Bubbling
document.addEventListener('DOMContentLoaded', () => { init(); // DOM ist bereit });
Schneller als 'load' — CSS/Bilder müssen nicht warten
load
Seite vollständig geladen (inkl. Bilder/CSS)
kein Bubbling
window.addEventListener('load', () => { ... });
beforeunload
User verlässt Seite — Dialog möglich
kein Bubblingabbrechbar
window.addEventListener('beforeunload', (e) => { e.returnValue = ''; // zeigt Browser-Dialog });
visibilitychange
Tab sichtbar / versteckt
kein Bubbling
document.addEventListener('visibilitychange', () => { if (document.visibilityState === 'hidden') pauseVideo(); });
Fenster & Navigation (4)
resize
Fenster wird skaliert · innerWidth / innerHeight
kein Bubbling
⚡ Immer mit debounce() drosseln!
scroll
Seite / Element wird gescrollt
kein Bubbling
window.addEventListener('scroll', () => { if (window.scrollY > 100) nav.classList.add('sticky'); }, { passive: true });
popstate
Browser Vor / Zurück · e.state
kein Bubbling
Basis für clientseitiges Routing in SPAs
error
JS-Fehler tritt auf · globaler Catcher
kein Bubbling
window.addEventListener('error', (e) => { logToServer(e.message, e.filename, e.lineno); });
Medien — Audio / Video (5)
play / pause / ended
Wiedergabe startet / pausiert / endet
kein Bubbling
video.addEventListener('ended', () => playNext());
timeupdate
Position ändert sich (~4× pro Sekunde)
kein Bubbling
video.addEventListener('timeupdate', () => { bar.value = video.currentTime / video.duration; });
volumechange
Lautstärke oder Mute ändert sich
kein Bubbling
// e.target.volume · e.target.muted
Legende: bubbelt Event wandert zum Parent kein Bubbling Event bleibt am Element abbrechbar preventDefault() möglich · Tipp: { passive: true } bei Scroll/Touch für bessere Performance
